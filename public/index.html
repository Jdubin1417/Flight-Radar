⚙️ Settings

Add free API keys to unlock more data. Keys are saved on your own computer and applied instantly — no restart needed.

Get a free key at aisstream.io (sign up → create an API key).

Optional. Anonymous works but is rate-limited. A free account at opensky-network.org (Account → API clients) raises the limit ~10×.

Drag your OpenSky credentials .json here or click to browse — then click Save

Optional. Enables the 🌦️ Rain map overlay — modeled precipitation covering the whole world (radar only covers some regions). Free key at openweathermap.org (sign up → API keys).

🌐 Enable the Extended feed?

This adds extra aircraft from FlightRadar24, a third-party service. Because FR24 also aggregates satellite-based ADS-B, it can show flights over the open ocean and other remote areas that the free ground-receiver networks (OpenSky, adsb.lol, airplanes.live) physically can't see.

  • It uses an unofficial endpoint — there is no public API or key.
  • It may be rate-limited or stop working at any time without notice.
  • Use it for personal, non-commercial viewing only, and respect FlightRadar24's terms of service.

Everything else in the app keeps working if this feed is off or unavailable. You can turn it off anytime with the 🌐 Extended button.

🌍 Welcome to Flight-Radar

Live sea, sky & orbit on a 3D globe — every ship, plane and satellite you see is pulled from free public data. No paid subscription, no USB stick.

Where the data comes from

  • 🛰️
    Satellites — live orbits from Celestrak (ISS, Starlink, GPS…). No key needed.
  • ✈️
    Aircraft — ADS-B from OpenSky plus community receivers (adsb.lol, airplanes.live). Works with no key; a free OpenSky account raises the limits.
  • 🚢
    Ships — live AIS from aisstream.io. Needs a free key (add it in Settings).
  • 🌐
    Extended flights — optional third-party feed (FlightRadar24) that also carries satellite ADS-B, so it can show planes over the open ocean. Experimental.

The controls (top-left)

  • Show 🛰️ ✈️ 🚢 — turn each layer on/off. The numbers are live counts.
  • Trails 🛰️ ✈️ 🚢 — draw each object's recent path. (Click one object for its full flight/orbit path.)
  • Map — pick a base map (Satellite, Streets, Terrain, Atlas, Live clouds), toggle 🏷️ Labels, and add weather: 🌧️ Radar (real ground radar) or 🌦️ Rain (modeled, whole-world — needs a free key). 🌐 Extended adds the ocean-reaching third-party flights (shown in green).
  • Find — filter everything by callsign, type, airline, country, or satellite. Try typing iss.
  • Click any plane, ship or satellite for a full detail panel.
  • Camera (bottom-right) — ＋/－ zoom, 🧭 face north, ⌂ home view.
